# MERN Web App

ourfuture

## Purpose:

A communal to-do list to help envision a better future.

## Set Up:

```git clone git@github.com:christianmbonet/visionsofthefuture.git```

```cd visionsofthefuture```

```yarn```

To start server:

```yarn server```

To start front-end:

```cd client```

```yarn start```
